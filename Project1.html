A Unified Framework for Disease and Analgesic Prediction

This project is an innovative platform designed to provide comprehensive disease predictions, integrating multiple machine learning models for different types of illnesses:

  • K-Nearest Neighbors (KNN) was implemented to predict general illnesses based on patient symptoms.
  • Support Vector Machine (SVM) was applied to predict more specific conditions such as diabetes and heart disease.
  • A customized Xception model was used for skin disease classification, achieving an accuracy of 87%, showcasing the power of image-based deep learning techniques.

The project demonstrates a multidisciplinary approach, combining both traditional machine learning and advanced deep learning methods to tackle real-world healthcare challenges.

Evaluation of Indian EVs

This project leverages Artificial Intelligence and Machine Learning to evaluate the performance of electric vehicles (EVs) on Indian roads by analyzing user reviews:

  • Web scraping techniques were employed to collect data from various sources, which was then processed into structured fields for analysis.
  • The project utilized NLP with the RoBERTa model for sentiment analysis on user reviews, identifying insights and trends in customer feedback.

By integrating advanced data collection and processing techniques, this project provided valuable insights into the real-world performance of EVs in India.

Object Detection in Drone Technology

This project applied AI and ML techniques to enable drones to detect and classify objects in real-time:

  • Leveraged computer vision algorithms and sensors to identify and classify objects dynamically during flight.
  • Implemented a system to calculate and maintain safe distances from detected objects, ensuring smooth and collision-free navigation.

The project demonstrated the application of AI in enhancing the safety and efficiency of drone operations, making it a valuable solution for various industries relying on drone technology.

Loan Prediction Model

This project focuses on building a machine learning model to predict whether a loan application should be approved or rejected based on several factors such as the applicant's income, age, loan amount, employment status, and credit history. The objective is to help financial institutions assess loan risk more accurately and make informed lending decisions.

  • Classification Algorithms: Implemented and compared models like Logistic Regression, Decision Tree, and Random Forest to predict loan approvals.
  • Evaluation Metrics: Assessed model performance using metrics such as accuracy, precision, and recall.
  • Goal: To enhance predictive capabilities for loan defaults, thus aiding banks and financial institutions in reducing credit risk and improving decision-making.

This project demonstrates expertise in machine learning for the financial sector, with a focus on risk assessment and predictive analytics.
